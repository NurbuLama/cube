# Charter communications - Programming Assignment

## Getting Started

- Cube Application Assessment

1. Clone this repo
2. Change directory: `cd bin/`
3. Add execute permission: `sudo chmod 755 <*.sh>`
4. Start local: `./start.sh`
5. Execute demo: `./demo.sh`
